Module 2: Control Flow

Making Decisions and Repeating Actions

So far, our programs have run from top to bottom, one line at a time. But the real power of programming comes from controlling which lines of code are executed and how many times. This is called control flow, and it allows our programs to react to different situations and handle repetitive tasks efficiently.

1. Conditional Statements: `if`, `elif`, `else`

Conditional statements allow you to run certain blocks of code only if a specific condition is true. The fundamental building blocks are if, elif (short for "else if"), and else.

  • if: Checks a condition. If it's True, the code inside the block runs.
  • elif: If the first if was False, Python checks the elif condition. You can have multiple elif statements.
  • else: If all preceding if and elif conditions were False, the else block runs.

You can create complex conditions using logical operators: and, or, and not.


# A program to decide what to wear
temperature_c = 22
is_raining = False

if temperature_c > 25 and not is_raining:
    print("It's hot and sunny! Wear shorts and a t-shirt.")
elif temperature_c > 15 and is_raining:
    print("It's mild but raining. A light jacket and an umbrella are a good idea.")
elif temperature_c <= 15:
    print("It's cold. Wear a warm coat.")
else:
    print("Enjoy the mild weather!")

The Bigger Picture: AI and Decision Making

At a much more complex scale, AI models are all about making decisions. A spam filter uses conditions to decide "spam or not spam." A self-driving car uses conditions to decide "accelerate, brake, or turn." Understanding simple if statements is the first step toward understanding these complex decision-making systems.

2. Loops: Repeating Code

Loops are used to execute a block of code multiple times. This is essential for working with collections of data or performing a task until a condition is met.

for loops

Used for iterating over a sequence (like a list, a string, or a range of numbers). You use a for loop when you know how many times you want to repeat.


# Loop through a list of items
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(f"I like to eat {fruit}s.")

# Loop a specific number of times
for i in range(5):  # range(5) generates numbers 0, 1, 2, 3, 4
    print(f"This is loop number {i + 1}")

while loops

Used to repeat a block of code as long as a condition is True. You use a while loop when you don't know in advance how many repetitions are needed.

Debugging Challenge: The Infinite Loop

What is wrong with the following code? It's supposed to print numbers from 1 to 5. How would you fix it? What is the name for this common problem?


num = 1
while num <= 5:
    print(num)
    # Hint: Something is missing here! The variable 'num' is never changed.

This is called an infinite loop. To fix it, you must ensure the condition of the `while` loop can eventually become `False`. In this case, you need to increment `num` inside the loop by adding `num = num + 1` or `num += 1`.
