Module 1: NumPy and Pandas

The Workhorses of Data Science in Python

Welcome to the libraries that power nearly every data science and machine learning project in Python. NumPy provides the foundation for efficient numerical computation, while Pandas offers high-level data structures and analysis tools.

1. NumPy: Numerical Python

NumPy's core feature is the powerful N-dimensional array object, or ndarray. These arrays are homogeneous and fixed-size, which allows NumPy to perform mathematical operations on them much faster than with standard Python lists.

import numpy as np

# Create a NumPy array from a Python list
my_list = [1, 2, 3, 4, 5]
my_array = np.array(my_list)

# Perform a vectorized operation (much faster than a loop)
doubled_array = my_array * 2
print(doubled_array) # Output: [ 2  4  6  8 10]

# Create a 2D array (a matrix)
matrix = np.array([[1, 2, 3], [4, 5, 6]])
print(matrix.shape) # Output: (2, 3) -> 2 rows, 3 columns

The Bigger Picture: Data as Numbers

AI models, especially neural networks, understand only numbers. An image is just a matrix of pixel values. Text is converted into numerical vectors. Sound is a sequence of numbers representing amplitudes. NumPy is the fundamental tool for representing and manipulating this data efficiently.

2. Pandas: Python Data Analysis Library

Pandas introduces two primary data structures: the Series (a 1D labeled array, like a single column) and the DataFrame (a 2D labeled array, like a spreadsheet or SQL table). It's the ultimate tool for data cleaning, transformation, and analysis.

import pandas as pd

# Create a DataFrame from a dictionary
data = {
    'ProductID': ['A1', 'B2', 'A1', 'C3', 'B2'],
    'SaleAmount': [100, 150, 120, 200, 140],
    'Region': ['North', 'South', 'North', 'West', 'South']
}
df = pd.DataFrame(data)

# Inspect the data
print(df.head()) # Shows the first 5 rows

# Select a column (a Series)
sales = df['SaleAmount']
print(f"Total Sales: {sales.sum()}")

# Perform powerful grouping operations
sales_by_region = df.groupby('Region')['SaleAmount'].sum()
print(sales_by_region)

3. Connecting NumPy and Pandas

Pandas is built on top of NumPy. You can easily access the underlying NumPy array of a DataFrame or Series using the .values attribute. This interoperability is key to the Python data science ecosystem.

AI-Proof Exercise: Sales Data Analysis

You are given a CSV file (create a simple one yourself named `sales.csv`) with columns: `Date`, `Category`, `Product`, `Price`, `QuantitySold`.

  1. Load the CSV into a Pandas DataFrame.
  2. Create a new column called `TotalRevenue` which is `Price * QuantitySold`.
  3. Find the top-selling product category by total revenue.
  4. Calculate the mean, median, and standard deviation of the `Price`.

Challenge: Identify the day with the highest total revenue. You will need to convert the `Date` column to a datetime object first using `pd.to_datetime(df['Date'])` and then group by date.
