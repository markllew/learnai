Module 5: Math Break 1 - Foundational Logic & Algebra

Programming isn't just about writing code; it's about solving problems using logic. In this math break, we'll connect the mathematical concepts of logic and algebra to the programming constructs you've already learned. This foundation is critical for understanding how AI models "reason" about data.

1. Propositional Logic

This is the formal study of logical statements. It directly maps to if statements in Python. The statement IF P THEN Q is crucial. It means that if P is true, Q must also be true.

Consider this real-world example. If it is raining, then the ground is wet. But if the ground is wet, it doesn't necessarily mean it's raining (a sprinkler could be on). This is a common logical fallacy called "affirming the consequent."

2. Algebra and Functions
